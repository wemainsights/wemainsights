Su di noi


Cosa siamo, cosa facciamo

Wema Insights nasce con la missione di aiutare le realtà aziendali piccole, medie e grandi a snellire il lavoro d'ufficio tramite automazioni. Attraverso lo sviluppo di soluzioni software customizzate possiamo aiutare le organizzazioni a risparmiare tempo prezioso da poter impiegare in attività più produttive.

I nostri strumenti sono linguaggi di programmazione come Python e VBA. Possiamo automatizzare la compilazione di un foglio excel o fornire dati provenienti da un sito web giornalmente nel formato desiderato. La nostra visione è che Wema Insights può esserci dove non ha economicamente senso far intervenire un colosso della consulenza e allo stesso tempo non arrivano le risorse interne all'azienda.

Servizi

Macro personalizzate

Sviluppiamo le tue macro secondo le tue esigenze. A seguito di una consulenza specializzata scriveremo il codice che necessiti per automatizzare le tue attività noiose con Excel. Che si tratti di data entry o di elaborazione del dato già immesso noi possiamo intervenire.

Web scraping

Se la tua attività fa molto uso di dati presi dal web giornalmente probabilmente sarebbe più semplice effettuarlo tramite un'automazione. Se il sito web lo permette possiamo fare in modo che quei dati vengano archiviati in un file ed inviati giornalmente al vostro indirizzo di posta elettronica, pronti per essere analizzati.

File management

In alcuni casi, all'interno delle organizzazioni, uno dei compiti giornalieri di alcune mansioni può essere quello di spostare file da una cartella all'altra. Ciò può essere davvero un'attività tediosa. In realtà lasciar fare questo genere di cose al computer è di solito molto semplice (ed è anche la scelta migliore).
